Library Management

This is a Library Management Application built with JavaFX and SceneBuilder. The database is hosted with XAMPP and the queries are made using SQL code. The application creates "Book" objects and inserts them in a database. Later on, if a person wants to borrow a book they can do that by selecting the desired book that will be saved in the "Borrowed" table as a new object.

Client-Server Application

This is a game of tic-tac-toe between a player and the computer. The application is created with python and the server uses TCP as the transportation protocol. The whole game is a message exchange between the server and the client. It follows the basic rules of the game and whoever gets 3 symbols on line or diagonal wins.

Boosted();

C# Video Game built with Unity Engine. The props are hand-made and the game follows the simple platformer game rules. The player starts on a safe platform and has to proceed through the level without hitting any obstacle. The player wins the level by reaching the other safe platform. When the condition is met, the next level loads. The game ends when the player wins all levels.

Data Forecast

The program is based on time series. Given the initial information such as how many customers a store had in the previous months, the program can forecast future data, as how many customers the store will have in the next month/s. The program is created with python and uses the "Pandas" library along with the moving average as logics.
